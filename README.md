This project is created for Lab Class to practice building a modern React application using Vite + React + TypeScript.
The goal of the lab is to learn how to:

work with APIs

manage component state

store data in localStorage

build a clean UI with reusable components

use TypeScript for type-safe React development

This lab focuses on implementing a Movie Watchlist App, where students search movies, save them to a watchlist, and manage personal notes and ratings.
